# Mon Portfolio Nginx

Ce projet contient mon portfolio servi par un serveur Nginx.

## Comment lancer le projet
1. Assurez-vous d'avoir **Docker** installé.
2. Clonez le dépôt : `git clone https://github.com/brahus76/ibrahim-drabo-nginx.git'`
3. Lancez la commande : `docker build -t portfolio .`
4. Exécutez le conteneur : `docker run -p 8080:80 portfolio`
5. Accédez au site sur `http://localhost:8080`
